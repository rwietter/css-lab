Acessibilidade...

O conteúdo adicionado usando pseudoelementos não é inserido no DOM - é exibido apenas visualmente. Portanto, os leitores de tela não poderão acessar e ler o conteúdo gerado usando pseudoelementos. Portanto, é recomendável não usar pseudoelementos para inserir conteúdo vital em uma página (como notas de rodapé relevantes para o artigo, por exemplo).
